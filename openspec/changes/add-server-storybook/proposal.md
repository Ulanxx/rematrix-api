# Change: 添加 Server 端 Storybook 文档

## Why
为了更好地理解和维护 Rematrix Server 项目，需要创建一个全面的 Server 端 Storybook 文档，详细说明：
- 所有 API 接口的功能和使用方法
- 涉及的技术栈和架构设计
- 核心代码逻辑的执行顺序
- 关键功能点的详细说明
- 补充项目中的代码注释

这将帮助开发者快速上手项目，理解系统架构，并便于后续的维护和扩展。

## What Changes
- 创建 Server Storybook 文档结构
- 编写 API 接口详细说明
- 添加技术栈和架构介绍
- 梳理核心业务流程和代码逻辑
- 补充关键代码块的注释
- 创建交互式的文档展示

## Impact
- 新增文档能力：server-storybook
- 涉及代码：所有 Controller、Service、Workflow 文件
- 提升开发体验和项目可维护性
- 为新团队成员提供完整的技术文档
